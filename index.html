Complete Class Reference

REST Resource

Ws_VGP_RestResource

  • @RestResource urlMapping /api/v1/vgp/*
  • doGet, doPost, doPatch, doDelete
  • All methods delegate to WebServiceRouter.route
Configuration

WebServiceRouter

  • route: orchestrates full request lifecycle
  • writeResponse: serialises ApiResponse to RestContext
  • Catches ApiNotFoundException, ApiBusinessException, ApiValidationException
Configuration

HandlerRegistry

  • getHandler(httpMethod, resourceType)
  • Queries API_Handler_Config__mdt custom metadata
  • Type.forName dynamic instantiation + cache
  • Throws ApiNotFoundException / ApiBusinessException
DTO

ApiRequest

  • fromRestContext: factory method
  • parseUrlSegments: version, resource, subResource
  • getBody(Type): delegates to RequestBodyParser
  • generateRequestId: 32-char hex
Configuration

QueryOptions

  • fields, filters, orderBy, orderDir, pageSize, pageNumber, offset
  • extractFilters: non-reserved params become filters
  • Throws ApiValidationException for invalid page_size
Configuration

RequestBodyParser

  • parse(Type): reads RestContext body
  • JSON.deserialize to IRequestDto, auto-calls validate
  • Throws ApiValidationException on empty or invalid body
Interface

IRequestHandler

  • handle(ApiRequest) returns ApiResponse
Handler - Abstract

BaseRequestHandler

  • Template Method: preProcess, execute, postProcess
  • Catches: ApiValidationException 400, ApiNotFoundException 404, ApiBusinessException 422, Exception 500
  • ApiLogger in finally block
Handler

Ws_VGP_GetVGPHandler

  • vgp-users → SiteUserService.getVGPUsers
  • get-cda → not implemented
  • Throws ApiNotFoundException on unknown subResource
Handler

Ws_VGP_PostVGPHandler

  • create-siteuser → SiteUserService.createSiteUser
  • Throws ApiNotFoundException on unknown subResource
Interface

ISiteUserService

  • getVGPUsers(ApiRequest): ApiResponse
  • createSiteUser(ApiRequest): ApiResponse
Service

SiteUserService

  • getVGPUsers: countAll + selectAll, paginated list
  • getActiveUser: countAllActiveUsers + selectActiveUser
  • createSiteUser: validate, dup check, gen key, hash pw, insert
  • Throws ApiBusinessException DUPLICATE_EMAIL
Interface

ISiteUsersSelector

  • selectSiteUsersByEmail, selectAll, selectActiveUser
  • countAll, countAllActiveUsers
Selector

SiteUsersSelector

  • extends fflib_SObjectSelector
  • Site_User__c: Id, Email__c, Priest__c
  • fflib_QueryFactory for dynamic SOQL
DTO

ApiResponse

  • success, statusCode, requestId
  • data (IResponseDto), errors (ApiError list), pagination
DTO

ApiError

  • code, message, field
Interface

IResponseDto / IRequestDto

  • IResponseDto: marker interface
  • IRequestDto: validate() method
Configuration

ApiResponseBuilder

  • ok: 200 single, okList: 200 paginated
  • error, notFound 404, forbidden 403, internalError 500
Configuration

PaginationMeta

  • build(totalRecords, opts)
  • totalRecords, pageSize, pageNumber, totalPages, hasNext, hasPrev
Response DTO

Response DTO

CreatedSiteUserDto

CreateSiteUserRequestDto

  • implements IRequestDto
